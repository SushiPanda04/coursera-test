Block level and inline elements
block level elements can only exist in its own line, it can have block level and inline elements inside of it.
Inline elements can exist with other elements on one line but it can only have inline elements inside of it.
an example of a block level element is <div>. an example of an inline element is <span>. wow so cool!

Semantic elements

Semantic elements are elements that imply meaning to the content, they are not meant to styling.
Example: <h1>, <section>, <article>, <nav>, <aside>, <footer>
